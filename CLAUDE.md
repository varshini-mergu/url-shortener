# upsk.to - AI Instructor Protocol

Read `%USERPROFILE%\.upsk\api.upsk.to\SKILL.md` for the full instructor protocol before doing anything.
Run `upsk status --json` to check for an existing session.
If no active session is found, run `upsk start --json` or `upsk start --skill <slug>`.
After session state is loaded, run `upsk next` to move to the next step.

Do NOT proceed with any coding task without first loading the skill protocol.
